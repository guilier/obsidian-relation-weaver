# 更新日志

## v2.0.0（2026年7月6日）

### 🎉 写作仪表盘发布

#### ✨ 新增
- 📊 **写作仪表盘**：数据概览、状态分布、待办提醒、时间线进度条
- 🎯 **时间感知状态**：根据「当前时间点」判断人物状态（存活/已故/未出生）
- 📌 **类型筛选**：按主角/配角/龙套筛选人物
- 🔍 **年份搜索**：时间线视图按年份/关键词搜索
- ⏱️ **当前时间点设置**：设置页新增配置
- 📑 **导出仪表盘报告**：一键生成 Markdown 报告
- 🖱️ **筛选横幅**：点击待办自动筛选并显示横幅

#### 🔧 改进
- 人物卡片显示彩色状态徽章
- 时间解析增强（支持括号年份格式 `（公元前221年）`）
- 优化界面交互体验
