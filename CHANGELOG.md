# 更新日志

## [3.0.0] - 2026-07-14

### 🎉 重大更新：插件正式迎来3.0版本

### ✨ 新增功能

#### 📊 增强型写作仪表盘
- **实时数据总览**：人物、事件、关系、阵营数量一目了然
- **智能状态分布**：基于当前时间点自动统计存活/已故/未出生/失踪/未知人数
- **一键跳转筛选**：点击「尚未出场」「无关系」「已故后仍出场」待办项，直接跳转并筛选人物列表
- **时间线进度条**：可视化展示故事覆盖进度（已覆盖百分比 + 时间跨度）
- **最近活动列表**：自动显示当前时间点之前的最新事件

#### 🔥 热力图（完整版）
- **三种模式**：
  - 集中视图：全人物出场频率一览
  - 单人深度：聚焦单个角色出场轨迹
  - 多人对比：同时对比 2-10 个角色的出场分布
- **粒度切换**：年 / 半年 / 季度
- **年份范围滑块**：自由拖拽选择时间区间
- **交互增强**：点击表头年份直达时间线，悬停数字弹出具体事件列表
- **分页支持**：人物较多时自动分页

#### 📈 亲密度动态演化系统
- **变化记录**：每次关系变化自动保存，形成完整历史
- **曲线可视化**：SVG 折线图直观展示亲密度趋势
- **三种记录方式**：
  - 📎 关联时间线事件（变化与剧情绑定）
  - ✏️ 自定义描述（自由记录变化原因）
  - 📌 初始值设定（新建关系时记录起点）
- **历史弹窗**：点击「📊 历史」查看完整变化列表，支持按时间排序
- **变化摘要**：每条关系卡片上直接显示「共X次变化，当前: XXX」

#### 🎨 使用场景预设系统
- **5 种预设场景一键切换**：
  - 小说创作：全功能布局
  - 日常日记：聚焦时间线和人物事件
  - 跑团 TRPG：专注角色与冒险
  - 知识图谱：突出概念关联
  - 自定义：自由开关 Tab
- **字段预设联动**：切换场景时自动适配阵营、关系类型、时间字段名称

#### 🏷️ 人物类型筛选
- 自动识别人物文件中的「类型」字段
- 智能排序：⭐主角 → 🔶配角 → 👤龙套 → 其他
- 自动添加 Emoji，视觉更清晰
- 与状态筛选、搜索、阵营筛选叠加使用

#### ⏱️ 当前时间点系统
- 设置故事当前时间点（支持公元前/公元格式）
- 人物状态自动判断：🟢存活 / 🔴已故 / 🔵未出生 / ⚪失踪 / 🟡未知
- 状态徽章展示在人物卡片上
- 状态筛选一键过滤

#### 📋 Tab 显示开关
- 每个 Tab 独立开关（设置页可视化操作）
- 配合场景预设自动适配
- 「人物」和「导入导出」Tab 始终显示

#### 🔍 时间线搜索增强
- 支持年份数字精确匹配
- 支持年份字符串模糊搜索（如「300」「前280」）
- 支持事件关键词搜索
- 搜索结果实时高亮

---

### 📌 升级说明

如果你是从 v2.1.x 升级到 v3.0.0：

1. **无需迁移数据**：数据存储路径自动适配新 ID，旧数据保留
2. **推荐更新方式**：通过 Obsidian 社区插件市场直接更新
3. **手动更新**：从 [Releases](https://github.com/guilier/relation-weaver/releases) 下载最新版本，替换 `.obsidian/plugins/relation-weaver/` 文件夹内容
